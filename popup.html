Choose Between Three Modes:
  1. Doc Page: Brings you to the company documents page and sort in descending order.
  2. Linkgrabbe: Consolidates links from multiple pages and let you download documents.
  3. Downloader: Will auto-download multiple documents. Note that downloading a large number of files will trigger CAPTCHA. Use sparingly. VPN recommended.

Enter from any field = Run.
Ctrl + Shift + Backspace = Reset
Hold Ctrl to select multiple Filing Types.

Using date field: Numeric date values can be YYYY, YYYYMM, or YYYYMMDD. You can also specify weeks, months or quarters. L5W / L5M / L6Q. Single YYYYMMDD value = from date. i.e. 20200101 will pull filings from that day onwards. Double YYYYMMDD value = from -> to. i.e. 20200101-20230101 will pull filings between 2020->2023.
